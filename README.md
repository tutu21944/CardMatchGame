# CardMatchGame
纸牌匹配消除游戏（Cocos2d-x 3.17）
项目简介
本项目基于 Cocos2d-x 3.17 开发，实现一款简单纸牌匹配消除小游戏，支持卡牌匹配、手牌替换、回退撤销三大核心功能。采用 MVC 架构，代码分层清晰、可维护、可扩展，完全遵循需求文档中的设计规范。
游戏玩法
• 主牌区：显示待消除的桌面纸牌
• 手牌区：显示当前底牌，可被桌面牌替换
• 匹配规则：数字差 1 即可匹配（无花色限制）
• 备用牌堆：可翻牌补充底牌
• 回退：支持多次撤销操作，卡牌自动回退到上一步位置
核心功能
1. 桌面牌与手牌匹配：点数 ±1 即可消除并替换手牌
2. 手牌区翻牌替换：点击手牌区卡牌，自动平移替换顶部牌
3. 回退（撤销）功能：记录每一步操作，支持无限回退
4. 关卡配置加载：JSON 配置文件动态生成关卡布局
5. 平滑动画：卡牌移动、替换、回退均带 MoveTo 动画

运行说明
1. 使用 Visual Studio 2022 打开项目
2. 配置好 Cocos2d-x 3.17 环境
3. 编译运行即可进入游戏
4. 点击桌面牌进行匹配；点击回退按钮撤销操作

扩展说明
• 新增卡牌：只需在 CardModel、CardView、配置表中扩展，无需改动核心逻辑
• 新增回退类型：在 UndoModel、UndoManager 中新增记录类型即可
• 新增关卡：直接编辑 JSON 关卡配置文件，自动加载布局
